Politique de Confidentialité – Aywa! et Aywa! Pro

Dernière mise à jour : 05 décembre 2025

La présente Politique de Confidentialité décrit comment Aywa! (pour les passagers) et Aywa! Pro (pour les chauffeurs) collectent, utilisent, stockent et protègent les informations des utilisateurs lors de l’utilisation des applications mobiles.

1. Informations que nous collectons

1.1 Informations fournies par l’utilisateur

Lors de l'inscription, nous collectons :

  • Numéro de téléphone
  • Nom et prénom (si fournis)
  • Code OTP envoyé par SMS
  • Pour les chauffeurs : informations professionnelles (véhicule, permis, etc.)

1.2 Informations collectées automatiquement

  • Identifiant du téléphone (Device ID)
  • Informations techniques (modèle, version Android/iOS)
  • Adresse IP et données de connexion

1.3 Géolocalisation

Pour permettre la réservation et la gestion des trajets, les applications peuvent collecter votre position GPS, ainsi que les points de départ et d’arrivée.

2. Utilisation des données

Nous utilisons vos données pour :

  • Créer votre compte utilisateur
  • Vérifier votre identité à l’aide d’un code OTP
  • Assurer la sécurité de l’accès à l’application
  • Permettre la réservation des trajets VTC
  • Pour les chauffeurs : gérer les courses et les informations professionnelles
  • Améliorer les services et l’expérience utilisateur

3. Partage des données

Nous ne vendons jamais vos données. Les données peuvent être partagées uniquement avec :

  • Le prestataire SMS pour l’envoi du OTP
  • Les chauffeurs et passagers nécessaires pour effectuer la course
  • Les autorités en cas d’obligations légales

4. Sécurité des données

Nous utilisons différentes mesures de sécurité (chiffrement, serveurs sécurisés, restrictions d’accès) pour protéger vos informations personnelles.

5. Conservation des données

Les données sont conservées uniquement le temps nécessaire pour fournir les services Aywa! et Aywa! Pro, et se conformer aux obligations légales.

6. Vos droits

Vous pouvez demander à :

  • Accéder à vos données
  • Modifier vos informations
  • Demander la suppression de votre compte

7. Contact

Pour toute question, veuillez nous contacter à :
Email : contact@aywa.com

8. Modifications

Cette Politique de Confidentialité peut être mise à jour à tout moment. La date de mise à jour sera toujours indiquée en haut du document.
